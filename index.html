MOSES GWAZA

PORTFOLIO

This is a portfolio with various projects by Moses Gwaza.The projects demonstrate the skills of using different tools to tranform raw data into useful insights with the core focus on solving a problem.

The projects use Excel and Google Sheets for smaller datasets, the Power Platform with particular focus on Power BI, programming languages R and Python from Webscraping to creation of Data Viz, SQL for data exploration and manipulation, as well as telling stories in form of charts and dashboards using Tableau.

2022 Malawi Cholera Dashboard

The Cholera Project in Tableau analyzes the 2022 Cholera outbreak in Malawi, focusing on data from February to August to identify key insights such as total cases, affected regions, and risk factors like unsafe water sources.

2020-2022 Covid-19 Dashboard

This project analyzes the WHO data of COVID-19 pandemic from January 2020 to November 2022 using Python to uncover trends and answer key questions. An R Shiny interactive dashboard showcases results.

Transforming OpenMRS HIV Data

This project implements an ETL pipeline to transform OpenMRS HIV patient data to a Common Data Model and performs patient-level analysis, visualizing the results. Uses SQL, Pentaho, PostgreSQL and MySQL Databases, OHDSI Tools.

Lung Cancer Predictive Analysis

This uses Kaggle Survey data for Descriptive and Predictive analysis using Python. Showcased results using a Decision Tree, a Funnel Chart and an Interactive Predictive Widget.

Sales Insights Project

A video demonstration of Power BI's interactive dashboard and a fully documented project - loads and explores data in MySQL, connects to Power BI, uses Query Editor, modelling and Dashboarding.

Excel Projects

Advanced compelling Dashboards were built for Suicide rates and Financial statements . Data Filters, Functions and Formulas, Charts and Plots, Pivot Tables, Form Controls were used.

Google Analytics Projects

This directory contains two projects by the Google Analytics Certificate Program. Smart Devices on Women's health using R and World Happiness Score in Tableau.

India House Predictions

Linear Regression in Python was used to train a model that takes house prices and tries to make a prediction . The dataset is of Bangalore India found on Kaggle.

Webscraping IMDB Movies

The primary components of this project are web scraping the top films from IMDB, cleaning and arranging the dataset into appropriate tables, and finally statistical and visual analysis of the data using Python code.
